# TpCpp

## Retrouvez ici deux de mes travaux pratiques en C++
Ces TP ont été réalisés en C++ sur QtCreator. Ils ont été réalisés en début d'année afin d'appréhender les design patterns.

## Titanic
Dans ce TP, nous chargeons un fichier contenant des données statistiques suite au naufrage du Titanic.
Ce TP m'a permis d'apprendre le design pattern MVC.

## Gestion d'étudiants
Ce TP charge une liste d'étudiants. Nous pouvons en supprimer, en ajouter, consulter des données liées aux étudiants et les modifier.
